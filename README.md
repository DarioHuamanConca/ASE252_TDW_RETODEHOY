# ASE252_TDW_RETODEHOY

Proyecto base con Flask (Python) y Tailwind CSS.

## Requisitos

- Node.js + npm
- Python con entorno virtual `.venv`

## Instalación

1. Instalar dependencias de Node:

   ```powershell
   npm install
   ```

2. Activar entorno virtual:

   ```powershell
   .\.venv\Scripts\Activate.ps1
   ```

## Ejecución

1. En una terminal, compilar Tailwind en modo watch:

   ```powershell
   npm run tailwind:build
   ```

2. En otra terminal, iniciar Flask (entrada principal por Python):

   ```powershell
   npm start
   ```
